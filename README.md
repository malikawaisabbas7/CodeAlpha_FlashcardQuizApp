✨ Features

🃏 Interactive Flashcards — Study with digital flashcards that flip to reveal answers
🔄 Card Flip Animation — Smooth 3D flip animation on every card interaction
✅ Quiz Mode — Test yourself and track correct/incorrect answers
📊 Score Tracking — View your quiz score at the end of each session
🎨 Material Design UI — Clean, modern interface following Google's Material guidelines
📱 Responsive Layout — Works perfectly on all Android screen sizes

🚀 How to Run

Prerequisites

Android Studio Hedgehog (2023.1.1) or newer
Android SDK API 24 or higher
Kotlin 2.0.0

Steps

bash# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/CodeAlpha_FlashcardQuizApp.git

# 2. Open in Android Studio
File → Open → Select the cloned folder

# 3. Sync Gradle
File → Sync Project with Gradle Files

# 4. Generate App Icons (if needed)
Right-click res/ → New → Image Asset → Finish

# 5. Run the app
Click ▶ Run or press Shift + F10
